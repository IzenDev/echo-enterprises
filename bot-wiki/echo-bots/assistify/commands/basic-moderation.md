Moderation Commands for Assistify are listed below:

* `mute` - Mute/Timeout a user for a set time.
* `unmute` - Unmute a muted user.
* `kick` - Kick a user from the server.
* `ban` - Ban a user from the server.
* `unban` - Unban a banned user by username, ID, or mention.
* `lock` - Lock the current channel, and stop new messages.
* `unlock` - Unlock the channel
* `help` - View the help menu for all commands.
